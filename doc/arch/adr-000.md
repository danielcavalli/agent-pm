## Organization

The ADRs are grouped by theme:

**Data & Storage (001-005):** How pm stores, structures, identifies, and indexes project data.

**Architecture & Interfaces (006-009):** How the four interfaces (CLI, MCP, slash commands, TUI) are layered and how they interact with the data layer.

**Implementation (010-014):** Technology choices for the implementation — language, validation, CLI framework, TUI framework, and error handling.

**Integration & Deployment (015-018):** How pm is installed, configured across multiple AI clients, and how agent behavior is shaped.

**Evolution (019-020):** Transitional decisions for migrating from earlier architectures.
